# Playback

Driven by `requestAnimationFrame`, independent of video element events.

## Controls

| Action | Shortcut |
|--------|----------|
| Play / Pause | `Space` |
| Frame step backward / forward | `←` / `→` |
| 1-second step | `Shift+←` / `Shift+→` |
| Jump to start | `Home` |
| Jump to end | `End` |

Timecode display shows current time and total duration in HH:MM:SS format.
