# Media

## Supported Formats

| Type | Formats |
|------|---------|
| Video | MP4, MOV, AVI, WebM, MKV |
| Audio | MP3, WAV, AAC, OGG, M4A |
| Image | PNG, JPG, GIF, WebP, SVG |
| Component | TSX, JSX |

## Import

Files are imported via the system file dialog and copied into the project's `media/` folder. Built-in components (ColorBackground, TextOverlay, Animation, CountdownTimer) can be added from a dropdown.

## Sidebar

- Click to preview in the preview panel (standalone mode).
- Drag onto a timeline track to place a clip.
- Double-click to add to the next available slot.
- Right-click to remove (also removes associated timeline clips).
- Each file has an expandable metadata field with per-type hints.
