# ColorBackground

Renders an animated gradient background that cycles through hues over time.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| startHue | number | 0 | Starting hue value (0â€“360) |
| endHueOffset | number | 60 | Hue offset for the second gradient color |

## Behavior

- Hue rotates 360 degrees over the full clip duration starting from `startHue`
- Gradient angle is 135 degrees
- Second color is offset by `endHueOffset` from the primary hue
