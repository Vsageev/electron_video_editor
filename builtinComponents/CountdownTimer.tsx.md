# CountdownTimer

Displays a countdown timer that counts down from the clip duration to zero.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| label | string | "" | Optional label text below the number |
| color | color | #ffffff | Text color |

## Behavior

- Shows remaining whole seconds (ceiling)
- Progress bar fills from right to left as time passes
- Dark radial gradient background

## Tags
