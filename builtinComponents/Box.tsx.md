# Box

Renders a colored rectangle with optional child content, shadow, and padding.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| color | color | #4a90d9 | Background color |
| borderRadius | number | 0 | Corner radius in pixels |
| padding | number | 0 | Inner padding in pixels |
| shadow | boolean | false | Drop shadow |
| child | media | (none) | Child media/component to render inside |

## Tags
#component #background #color #shape #container
